Welcome to Quisko

About

This is a small and simple Django application in which users can solve quizzes and if they like they can create their own quiz so friends and other quiz lovers can try to get the best score. To be able to create a quiz you have to make a simple registration.

Quisko application is made with Django Framework and a little bit of jQuery. Template is taken and modified from HTML Codex and some icons are taken from Freepik.com. If you like you can get source code from this Github repository. Enjoy in solving and creating questions.
